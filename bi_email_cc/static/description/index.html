Email CC BCC With Reply to Option Odoo Apps

Email CC BCC With Reply to Option Odoo App helps users to send email to multiple partners in CC and multiple partners in BCC of mail with specific reply to option. Users can select partners for CC, BCC and also make BCC or CC visible in emails by enabling them from general settings, according to the requirement.Also have option for Customer Email Reply-which helps to add Reply-To to every mail. User can set default Reply-To customer to mail from the general settings.

Features

Set Default CC & BCC for Email

User can set default CC and BCC for email in general settings.

Set Default Reply-To for Email

User can set default reply-to for email in general settings.

Send Mail to the Customers with CC & BCC

User can send mail to the customers with selected CC and BCC in general settings.

Add Reply-To the Mail

User can add reply-to the mail as per selected in general settings.
